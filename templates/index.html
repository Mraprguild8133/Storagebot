Wasabi Storage Bot

This service provides high-speed file storage and media playback through Telegram.

High-Speed Transfers: Optimized download and upload speeds with parallel processing.
Media Playback: Direct playback of video and audio files in your browser.
Secure Storage: All files are securely stored on Wasabi cloud storage.
Easy Access: Access your files anytime through Telegram commands.

To use this service, simply send files to the Telegram bot or use the commands:

  • /start - Show help information
  • /list - List your stored files
  • /download <filename> - Get a download link
  • /play <filename> - Get a player link for media files
